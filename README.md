# Demo UI Components

Workbench para desarrollar componentes React reutilizables.

## Estructura

- `src/components`: componentes exportables de la libreria.
- `src/workbench`: entorno visual local para probar estados y variantes.
- `src/index.ts`: API publica del paquete.

## Comandos

```bash
npm install
npm run dev
npm run typecheck
npm run build
```

El workbench se abre con Vite y la build genera una libreria en `dist`.
